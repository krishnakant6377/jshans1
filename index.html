Q.3) You are given two numbers "A" and "B".
You need to do the following checks:
(i) if both are divisible by 10 console true.
(ii) if both are not divisible by 10 console false.
(iii) if one of them only is divisible by 10 console true.
Use operators to do this.

Ans:
const Check = (A, B) => {
if (A % 10 == 0 && B % 10 == 0){
return true;
} else if (A % 10 != 0 && B % 10 != 0){
return false;
}
else {
return true;
}
};
